# Godot Messenger

Кроссплатформенный мессенджер с поддержкой текстовых сообщений, файлов и голосовых/видеозвонков через WebRTC, написанный на Godot 4.

## 🚀 Возможности

- 🔐 **Аутентификация пользователей** — регистрация и вход через сервер.
- 💬 **Личные сообщения** — отправка текстовых сообщений выбранному пользователю.
- 📁 **Отправка файлов** — изображения, видео, PDF (до 5 МБ). (Не работает)
- 📞 **Аудио/видеозвонки** — одноранговые соединения через WebRTC. (Не работает)
- 🟢 **Статус "онлайн/оффлайн"** — отображается в списке контактов.
- 🧠 **История сообщений** — загрузка предыдущих диалогов с пользователем.
- 🔍 **Поиск пользователей** — быстрый поиск по логину.

> ⚠️ Проект находится в статусе "исторического" — я делал его давно, но он отлично подходит для изучения сетевого программирования в Godot.

## 🧱 Технологии

| Компонент | Технология |
|-----------|-------------|
| Клиент | Godot 4 (GDScript) |
| Сеть клиента | WebSocketPeer |
| Звонки | WebRTCPeerConnection + WebRTCDataChannel |
| Серверная часть (ожидается) | Python + FastAPI (WebSocket + REST) |
| Протокол обмена | JSON over WebSocket |
| Хранение | Серверное (PostgreSQL / SQLite on backend) |

## 🛠 Сборка и запуск

### Требования
- Godot 4.2 или новее
- Серверная часть (должна быть запущена отдельно)
